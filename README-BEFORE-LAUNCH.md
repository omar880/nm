# قبل ما تنشر الموقع — قائمة سريعة

## 1) استبدل النطاق الوهمي
النطاق `yoursite.com` مستخدم في هذه الملفات لأغراض الـ SEO (canonical, sitemap, robots.txt, og:url, og:image):
- ابحث عن `yoursite.com` في كل ملفات المشروع (`grep -rl "yoursite.com" .`)
- استبدله بنطاقك الحقيقي (مثال: `اعمل find & replace بمحرر الأكواد من yoursite.com إلى mydomain.com`)

## 2) بريد نموذج التواصل
افتح `pages/contact.html`، ولاقي هذا السطر قرب نهاية الملف:
```js
var CONTACT_EMAIL = 'hello@yoursite.com';
```
غيّره لبريدك الحقيقي.

> ملاحظة: النموذج حاليًا يفتح تطبيق البريد عند الزائر (mailto:) لأن الموقع بدون سيرفر خلفي (backend).
> إذا حبيت الرسائل توصلك مباشرة بدون ما يفتح المستخدم بريده، بتحتاج خدمة خارجية بسيطة زي Formspree أو Getform (مجانية بحدود معينة) وتربطها بالفورم بدل الـ mailto.

## 3) تأكد إنك رفعت مجلد الصور الجديد
`assets/images/` فيه:
- `favicon.ico`, `favicon-16.png`, `favicon-32.png`, `favicon-180.png`, `favicon-512.png`
- `og-image.png` (صورة المشاركة على واتساب/فيسبوك/تويتر)

تأكد هاي الملفات انرفعت مع باقي الموقع.

## 4) اختياري: صور مشاركة مخصصة لكل نمط
حاليًا كل صفحات الأنماط الثمانية تستخدم نفس صورة المشاركة العامة (og-image.png).
لو حبيت لاحقًا صورة مختلفة لكل نمط (SPL، ICE...) بترفع تفاعل المشاركة أكتر، قلي وأسويها.

## 5) تفعيل ميزة "اسأل الذكاء الاصطناعي عن نتيجتك" (Vercel)

أضفنا دردشة حقيقية بالذكاء الاصطناعي تظهر بصفحة النتيجة. تحتاج خطوتين بسيطتين على Vercel:

### أ) احصل على مفتاح Anthropic API
من https://console.anthropic.com/settings/keys — أنشئ مفتاح جديد وانسخه.

### ب) ضيف المفتاح كمتغير بيئة على Vercel
1. افتح مشروعك بلوحة تحكم Vercel
2. Settings → Environment Variables
3. أضف متغير باسم `ANTHROPIC_API_KEY` وقيمته المفتاح يلي نسخته
4. اختر البيئات (Production + Preview) واحفظ
5. أعد نشر المشروع (Redeploy) حتى يتفعّل المتغير الجديد

هيك بس — الملف `api/ask-ai.js` جاهز وبيشتغل تلقائيًا كـ Serverless Function بمجرد ما ترفع المشروع، بدون أي إعداد إضافي (Vercel بيتعرف على مجلد `api/` تلقائيًا).

### ملاحظات مهمة
- الكود يستخدم نموذج `claude-haiku-4-5-20251001` (سريع واقتصادي، مناسب لهيك استخدام). لو حبيت جودة أعلى بتكلفة أكبر شوي، غيّرها بـ `claude-sonnet-5` بملف `api/ask-ai.js`.
- فيه حد أقصى 8 أسئلة لكل زائر بالجلسة الواحدة (من طرف المتصفح)، بالإضافة لتحديد بسيط لعدد الطلبات بالدقيقة من نفس الـ IP (طرف السيرفر). هاي حماية أساسية بس مش قوية 100% — إذا توقعت زيارات كتيرة وحابب حماية أقوى من إساءة الاستخدام (وبالتالي فاتورة أعلى)، فكّر تضيف Vercel KV أو Upstash Ratelimit لاحقًا.
- كل سؤال بيكلفك تقريبًا جزء بسيط جدًا من السنت الواحد (نموذج Haiku رخيص جدًا)، بس تابع استهلاكك من https://console.anthropic.com/settings/usage خصوصًا بالأسابيع الأولى بعد النشر.
