# قبل ما تنشر الموقع — قائمة سريعة

## 1) استبدل النطاق الوهمي
النطاق `yoursite.com` مستخدم في هذه الملفات لأغراض الـ SEO (canonical, sitemap, robots.txt, og:url, og:image):
- ابحث عن `yoursite.com` في كل ملفات المشروع (`grep -rl "yoursite.com" .`)
- استبدله بنطاقك الحقيقي (مثال: `اعمل find & replace بمحرر الأكواد من yoursite.com إلى mydomain.com`)

## 2) بريد نموذج التواصل
افتح `pages/contact.html`، ولاقي هذا السطر قرب نهاية الملف:
```js
var CONTACT_EMAIL = 'hello@yoursite.com';
```
غيّره لبريدك الحقيقي.

> ملاحظة: النموذج حاليًا يفتح تطبيق البريد عند الزائر (mailto:) لأن الموقع بدون سيرفر خلفي (backend).
> إذا حبيت الرسائل توصلك مباشرة بدون ما يفتح المستخدم بريده، بتحتاج خدمة خارجية بسيطة زي Formspree أو Getform (مجانية بحدود معينة) وتربطها بالفورم بدل الـ mailto.

## 3) تأكد إنك رفعت مجلد الصور الجديد
`assets/images/` فيه:
- `favicon.ico`, `favicon-16.png`, `favicon-32.png`, `favicon-180.png`, `favicon-512.png`
- `og-image.png` (صورة المشاركة على واتساب/فيسبوك/تويتر)

تأكد هاي الملفات انرفعت مع باقي الموقع.

## 4) اختياري: صور مشاركة مخصصة لكل نمط
حاليًا كل صفحات الأنماط الثمانية تستخدم نفس صورة المشاركة العامة (og-image.png).
لو حبيت لاحقًا صورة مختلفة لكل نمط (SPL، ICE...) بترفع تفاعل المشاركة أكتر، قلي وأسويها.
